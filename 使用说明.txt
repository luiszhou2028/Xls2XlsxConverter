===============================================
        XLS到XLSX批量转换器 v1.0 使用说明
===============================================

【简介】
这是一个基于Microsoft Office组件开发的Windows桌面应用程序，
用于批量将XLS文件转换为XLSX格式。

【系统要求】
- Windows 7 或更高版本
- Microsoft Excel 2007 或更高版本
- .NET 6.0 运行时环境

【安装步骤】
1. 确保已安装Microsoft Excel
2. 双击运行"编译程序.bat"进行编译
3. 编译成功后，双击"运行程序.bat"启动程序

【使用方法】
1. 【选择输入文件夹】
   - 点击"浏览"按钮选择包含XLS文件的文件夹
   - 程序将自动扫描该文件夹中的所有XLS文件

2. 【选择输出文件夹】
   - 点击"浏览"按钮选择保存XLSX文件的目标文件夹
   - 建议选择与输入文件夹不同的位置

3. 【设置转换选项】
   - ☑️ 包含子文件夹：勾选后会递归处理所有子文件夹
   - ☑️ 覆盖已存在的文件：勾选后会覆盖同名的XLSX文件

4. 【开始转换】
   - 点击"开始转换"按钮启动批量转换
   - 可通过进度条查看转换进度
   - 日志窗口会显示详细的转换信息

5. 【取消操作】
   - 转换过程中可点击"取消"按钮停止操作

【功能特点】
✓ 批量处理：一次转换多个XLS文件
✓ 保持结构：维持原有的文件夹层次结构
✓ 进度显示：实时显示转换进度和状态
✓ 详细日志：记录每个文件的处理结果
✓ 安全选项：可选择是否覆盖现有文件
✓ 取消支持：支持中途取消转换操作

【注意事项】
1. 转换过程中请勿手动打开或关闭Excel程序
2. 大文件转换可能需要较长时间，请耐心等待
3. 确保有足够的磁盘空间存储转换后的文件
4. 建议在转换前备份重要文件
5. 如果转换失败，请检查：
   - Excel是否正常安装
   - 源文件是否损坏
   - 是否有足够的系统权限

【常见问题】
Q: 提示"无法启动Excel应用程序"？
A: 请确保已正确安装Microsoft Excel，并尝试以管理员身份运行程序。

Q: 转换速度很慢？
A: 这是正常现象，Excel组件转换需要时间。避免同时处理过多大文件。

Q: 部分文件转换失败？
A: 检查源文件是否损坏，或者文件是否被其他程序占用。

【技术支持】
如遇到问题，请检查：
1. 系统是否满足运行要求
2. Excel是否能正常启动
3. 文件路径中是否包含特殊字符
4. 是否有足够的系统资源
